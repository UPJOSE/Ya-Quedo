## Definition of Done (DoD) del Sprint 3

Una tarea es **Done** cuando cumple:
1. Codigo mergeado a `develop` via Pull Request aprobado por al menos 1 reviewer
2. Conventional Commits (`feat:`, `test:`, `docs:`)
3. Backend: tests `BUILD SUCCESS` sin fallos
4. Frontend: arranca con `docker compose up` sin errores y consume API correctamente
5. Trazabilidad explicita entre User Story y task/endpoint/componente
6. Documentado en este backlog con estado actualizado
